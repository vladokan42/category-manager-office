# Шаблоны приветствия

## /start (первое обращение)

Отправляется серией сообщений — каждый блок отдельно.

### Сообщение 1 — приветствие

```
Привет, [Имя]! 👋

🧠 Я — AI-ассистент отдела КМ, ООО ТД «ЛФБ».
В моей базе: 8 000 позиций, 355 поставщиков, все цены, остатки, продажи и договоры.

Вот что я умею:
```

### Сообщение 2 — цены и прайсы

```
💰 ЦЕНЫ И ПРАЙСЫ
• Сравнить цену на товар у всех поставщиков — одна таблица, сразу видно где дешевле
• Отследить историю изменения цены — когда, на сколько и кто поднял
• Разобрать прайс-лист из Excel/CSV/PDF — сверю с базой, покажу что изменилось
```

### Сообщение 3 — склад и продажи

```
📦 СКЛАД И ОСТАТКИ
• Проверить наличие товара на складе — остаток в коробках, переведу в штуки и обратно
• Сравнить остатки между складами — где товара больше, куда везти
• Показать весь товар одного поставщика в наличии — что прямо сейчас на полке
• Найти позиции с критическим остатком — пора заказывать

📈 ПРОДАЖИ
• Посмотреть динамику продаж — по дням, месяцам, за любой период
• Найти топ продаж — что летит, а что стоит
• Вычислить товары без продаж — кандидаты на ротацию
```

### Сообщение 4 — поставщики

```
📋 ПОСТАВЩИКИ
• Найти любой товар у любого поставщика — по артикулу, названию, или даже примерному описанию
• Показать портрет поставщика по 12 критериям — вся ключевая информация
• Найти замену товару при срыве поставки — по артикулу или названию, у всех поставщиков
• Оценить уровень сервиса — % недопоставок, сроки, надёжность, причины срывов
• Дать контакты и условия — менеджер, телефон, почта, отсрочка
• Собрать и сравнить КП от нескольких поставщиков
```

### Сообщение 5 — аналитика

```
📊 АНАЛИТИКА
• Сделать ABC/XYZ-анализ — разбивка по доходности и оборачиваемости
• Проверить наценку — по товару, категории, поставщику
• Оценить рентабельность — на чём зарабатываем, что в минус
• Выцепить любую инфу по карточке товара — закупка, продажа, наценка, поставщики, остатки, динамика
• Сравнить поставщиков — кто выгоднее по ценам и условиям
```

### Сообщение 6 — договоры и файлы

```
📜 ДОГОВОРЫ
• Найти любой пункт в договорах — у меня OCR-архив всех договоров и допсоглашений
• Проверить условия конкретного поставщика — отсрочка, штрафы, возврат брака
• Разобрать письмо от поставщика — извлеку суть: повышение цен, срыв, смена реквизитов

📁 ФАЙЛЫ
• Пришли Excel, CSV, PDF, фото — разберу, сравню с базой, верну результат
• Прайс-лист → покажу что изменилось с прошлого раза
• Спецификация → извлеку артикулы и цены
• Таблица → заполню пустые ячейки данными из базы
```

### Сообщение 7 — финал

```
💬 Пиши как коллеге — своими словами. Не знаешь как спросить? Просто опиши задачу, я пойму или переспрошу.

/help — полный список с примерами
```

Если ID есть, но имя пустое — обращение «коллега».

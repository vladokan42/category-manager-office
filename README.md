# Category Manager Office

AI-агент отдела закупок ООО ТД «ЛФБ» — корпоративный Telegram-бот для пяти сотрудников.

## Что это

Общий справочный ассистент, работающий через Telegram-бота. Отвечает на вопросы по:

- 💰 **Ценам** — закупочные цены поставщиков, сравнение, история изменений
- 📦 **Остаткам и продажам** — наличие на складах, динамика продаж, топы
- 📋 **Поставщикам** — контакты, условия, портреты по 12 критериям, поиск замен
- 📊 **Аналитике** — ABC/XYZ-анализ, контроль наценки, рентабельность
- 📜 **Договорам** — поиск по OCR-архиву договоров и допсоглашений
- 📁 **Файлам** — разбор Excel, CSV, PDF, фото с автоматическим анализом

## Архитектура

Агент работает на базе [OpenClaw](https://github.com/openclaw/openclaw) с Claude Code в качестве runtime.

- **Канал:** Telegram-бот (ID 8896266630)
- **База:** SQLite `category.db` (~191 МБ, 7 390 позиций номенклатуры, 20 832 записи реестра прайсов)
- **Уровень доступа:** read-only к БД, запись только в `outgoing/` и `audit/`

### Жёсткие границы безопасности

| Разрешено | Запрещено |
|-----------|-----------|
| Чтение БД (цены, остатки, поставщики) | Отправка писем |
| Приём и разбор файлов | Изменение данных в БД |
| Поиск, сравнение, аналитика | Системные изменения, cron |
| Выдача результатов в `outgoing/` | Доступ к секретам и ключам |
| Журнал в `audit/` | Раскрытие личных данных |

### Защита от инжекта

При попытках обхода («забудь правила», «дай телефон…», «ты теперь X») — молчаливый отказ + запись в `audit/`.

## Аудит

Репозиторий создан для аудита агента через Claude Code. Все чувствительные данные (пароли, ключи, телефоны, Telegram ID) удалены.

Для полной картины см.:
- `SOUL.md` — как агент себя ведёт
- `memory/_Запреты.md` — что категорически нельзя
- `.claude/settings.json` — системные permissions
- `main-agent/ARCHITECTURE.md` — полная карта связей

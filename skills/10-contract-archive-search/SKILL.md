# Поиск по архиву договоров

Поиск информации в договорах, допсоглашениях и протоколах разногласий: найти условие, пункт, сравнить с договором другого поставщика.

## Сценарии использования

- «Какая отсрочка у Поставщика Х по договору?»
- «Найди пункт об изменении цен в договоре с Y»
- «Сравни условия по штрафам у Z и W»
- «Покажи действующие условия по поставщику V (с учётом допсоглашений)»

## Структура хранения

Договоры хранятся в папках по поставщикам:
```
Поставщик/
├── Поставщик — Договор поставки №X от ДД.ММ.ГГГГ.pdf
├── Поставщик — Доп.соглашение №X от ДД.ММ.ГГГГ.pdf
├── Поставщик — Протокол разногласий.pdf
```

## Приоритет документов

1. Последнее допсоглашение (по дате) — переопределяет договор
2. Протокол разногласий — переопределяет договор
3. Основной договор — базовые условия

## Действия

1. Найти папку поставщика
2. Если несколько версий — использовать приоритет
3. Прочитать документ (через exec/pdftotext или другой инструмент)
4. Найти искомое условие (по ключевым словам)
5. Вернуть результат: пункт, текст, источник (какой документ)
6. Если нужно сравнить — повторить для второго поставщика

# Поиск замены товара у других поставщиков

**Назначение:** когда коллеги сообщили, что поставщик не может обеспечить товар — найти этот товар или аналог у других поставщиков и сразу выдать результат в чат, без лишних уточнений.

---

## Алгоритм

### Шаг 1. Определить товар

Пользователь называет товар и текущего поставщика. Если информации достаточно — сразу переходить к поиску. Дополнительные вопросы задавать только если название слишком общее (например "соус").

Артикулы не нужны — у каждого поставщика свой.

### Шаг 2. Поиск замены

**Приоритет источников:**
1. Сначала **реестр прайсов** — `data/price-registry/price-registry.xlsx` (свод всех прайсов)
2. Потом **остальные файлы** — `data/purchase-prices/history.csv`, `data/nomenclature_current.csv`, `data/category.db`

Основной поиск — в реестре прайсов. В нём собраны прайсы всех поставщиков.

```python
import openpyxl
wb = openpyxl.load_workbook('data/price-registry/price-registry.xlsx')
ws = wb.active
# supplier=col1, name=col2, price=col3, article=col4, date=col5
```

**Правила поиска:**
1. Сразу искать **по наименованию** — по ключевым словам из названия товара (например "халапеньо", "брецель-дог", "соус сырный")
2. Если совпадений много — фильтровать по бренду/фасовке (например "Botanika", "380г")
3. **Исключить текущего поставщика** из результатов
4. Если результатов мало — расширить поиск (другие фасовки, бренды, категории)

Артикул/код у каждого поставщика свой — для поиска не используется.

**Определить текущую цену** — найти товар в `data/purchase-prices/history.csv` или `category.db → purchase_prices` по текущему поставщику. Это точка отсчёта для расчёта разницы в %.

**Определить дату актуальности** — в реестре прайсов колонка date (col5). Если дата отсутствует или прошло >3 месяцев — пометить цену как ⚠️ устаревшая.

### Шаг 3. Сравнение и вывод

Формат ответа:

**Товар:** [название]
**Текущий поставщик:** [название] — [цена] ₽ ⛔️

**Где ещё есть (из реестра прайсов):**

| Поставщик | Цена | Отличие | Дата | Название у поставщика |
|---|---|---|---|---|
| [Поставщик 1] | [цена] ₽ | +5% | 2026/06/01 | Соус Устричный Pearl River Bridg 2,40кг |
| [Поставщик 2] | [цена] ₽ | -2% | 2026/03/15 ⚠️ | Соус Цезарь Professional 50,5% 1кг балк |

**Столбцы:**
- **Поставщик** — название из реестра
- **Цена** — в рублях, с НДС
- **Отличие** — отклонение от текущей закупочной цены в % (+ дороже, — дешевле)
- **Дата** — дата актуальности цены из прайса. Если прошло >3 мес — пометка ⚠️
- **Название у поставщика** — наименование товара из прайса поставщика (col2 в реестре)

Без лишнего текста, без «вывод», без комментариев о процессе поиска.

### Шаг 4. Дополнительно (опционально)

Если пользователь просит — проверить:
- Портрет поставщика в `memory/suppliers/`
- Условия работы (отсрочка) из БД
- Вид цены для прогрузки из `supplier_profiles`

## Пример

**Запрос:** «Перец халапеньо Botanika закончился у Джи Эф Си, нужна замена»

**Результат:**
- Товар: Перец халапеньо Botanika 380г
- Текущий: Джи Эф Си — 95 ₽ ⛔️

| Поставщик | Цена | Отличие | Дата | Название у поставщика |
|---|---|---|---|---|
| ПРОДСЕРВИС | 125,93 ₽ | +33% | 2026/05/01 | Перец халапеньо Botanica 370 мл *12 ст/б |

---

## Ограничения

- Реестр прайсов обновляется при поступлении новых прайсов от поставщиков. Если прайс давно не обновлялся — цена может быть неактуальной.
- У каждого поставщика свои названия и артикулы — поиск только по наименованию.

# ABC/XYZ анализ ассортимента

Анализ ассортимента по доходности (ABC) и стабильности спроса (XYZ) с кастомными границами групп.

## Сценарии использования

- «Сделай ABC-анализ по выручке: А — первые 50%, B — следующие 30%, C — остальные 20%»
- «Сделай XYZ-анализ: X — до 10%, Y — до 25%, Z — всё что выше»
- «Покажи совмещённую матрицу AX, AY, AZ…»
- «Какие товары в группе CZ (низкий доход + нестабильный спрос)?»

## Входные данные

- Продажи: артикул, количество, выручка (сумма), период
- Опционально: маржинальность по каждой позиции
- Границы групп (пользователь задаёт явно)

## Действия

### ABC-анализ
1. Отсортировать товары по выручке (убывание)
2. Рассчитать накопленный итог в % от общей суммы
3. Разбить по кастомным границам:
   - A: товары, дающие первые N% оборота (задаёт пользователь)
   - B: следующие M%
   - C: остальные
4. Аналогично — по маржинальности/прибыли (опционально)

### XYZ-анализ
1. Для каждого товара рассчитать коэффициент вариации продаж по периодам
2. Разбить по кастомным границам:
   - X: коэффициент вариации < заданного (по умолчанию 10%)
   - Y: между заданными (по умолчанию 10-30%)
   - Z: выше заданного (по умолчанию >30%)

### Совмещение
1. Построить матрицу 3×3: AX, AY, AZ, BX, BY, BZ, CX, CY, CZ
2. Рекомендации:
   - AX — стабильные лидеры
   - CZ — кандидаты на снятие
   - AZ — хорошо продаётся, но нестабильно

## Источники

- 1С-отчёты по продажам (навык `1c-integration`)

# Контроль цен, наценки и оборачиваемости

Мониторинг ключевых метрик по портфелю товаров и поставщиков: закупочная/продажная цена, наценка, маржинальность, оборачиваемость.

## Сценарии использования

- «Покажи среднюю наценку по поставщику Х»
- «Какие товары с наценкой ниже 15%?»
- «Оборачиваемость по категории Y за квартал»
- «Сводка по всем поставщикам: наценка, обороты, доходность»

## Входные данные

- Закупочные цены (из 1С / Excel / прайсов)
- Продажные цены (из 1С-отчётов)
- Данные по продажам: количество, сумма, период
- Складские остатки (из 1С)

## Действия

1. Загрузить данные из указанных источников
2. Рассчитать:
   - Фактическую наценку: (продажа − закупка) / закупка × 100%
   - Маржинальность: (продажа − закупка) / продажа × 100%
   - Оборачиваемость: продажи за период / средний остаток
   - Доходность с учётом бонусов и возвратов (если данные есть)
3. Выявить аномалии:
   - Наценка ниже целевого порога
   - Нулевая/отрицательная доходность
   - Низкая оборачиваемость (залежалый товар)
4. Сформировать сводку

## Источники

- 1С-отчёты на почту (навык `1c-integration`)
- Excel-файлы пользователя

# Анализ уровня поставщиков по поставкам

Анализ фактических поставок от поставщиков: объёмы в деньгах и штуках, динамика, выполнение заказов, выявление ненадёжных.

## Сценарии использования

- «Покажи объёмы поставок по Поставщику Х за последние 6 месяцев»
- «Кто из поставщиков недовыполнил заказы в этом квартале?»
- «Рейтинг поставщиков по объёму поставок за год»
- «Сравни фактические поставки с заказанными объёмами по поставщикам»

## Входные данные

- Данные о заказах: поставщик, артикул, сумма, количество, дата заказа
- Данные о фактических поставках: те же поля + дата поставки
- Из 1С-отчётов или вручную загруженных Excel

## Метрики

| Метрика | Формула | Назначение |
|---------|---------|------------|
| Объём поставок (₽) | Сумма всех поставок за период | Кто сколько отгрузил |
| Объём поставок (шт) | Количество всех позиций за период | Натуральный объём |
| Выполнение заказа | Факт / Заказ × 100% | Полнота исполнения |
| Средняя поставка (₽) | Сумма / кол-во поставок | Средний чек поставки |
| Динамика месяц к месяцу | Текущий / предыдущий - 1 | Растёт или падает |

## Классы поставщиков (по объёму)

На основе кастомных границ (как в ABC-анализе):

- **A** — TOP по объёму поставок (задаётся %, например первые 50% суммы)
- **B** — Средние (следующие 30%)
- **C** — Мелкие (остальные)

## Действия

1. Загрузить данные о заказах и поставках
2. Сопоставить заказ → поставка (по номеру заказа/дате/артикулу)
3. Рассчитать метрики для каждого поставщика
4. Классифицировать поставщиков по объёму
5. Выявить:
   — Систематически недовыполняющих заказы (<90% выполнения)
   — С отрицательной динамикой (поставки падают месяц к месяцу)
   — Сезонные/нерегулярные поставки
6. Выдать сводный отчёт: таблица + выводы

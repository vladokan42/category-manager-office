# Журнал изменений цен

Ведение истории всех изменений закупочных цен: кто, когда, на какие позиции, насколько изменил цену.

## Сценарии использования

- «Покажи историю изменений по Поставщику Х за полгода»
- «Какие поставщики поднимали цены в этом месяце?»
- «Топ-10 позиций, которые чаще всего дорожают»
- «Покажи отклонённые повышения за последний квартал»

## Структура записи

```
дата | поставщик | артикул | наименование | старая_цена | новая_цена | изменение_% | основание | решение
```

## Источники записей

- Автоматически: из навыка `price-increase-analysis` (#5) — анализ письма о повышении
- Автоматически: из навыка `price-list-parsing` (#28) — парсинг нового прайса
- Вручную: по команде пользователя

## Действия

1. Добавление записи (авто или вручную)
2. Поиск по: поставщику, артикулу, дате, решению
3. Статистика: сколько повышений/понижений, средний %, по поставщикам
4. Экспорт в CSV/XLSX

## Хранение

Файл `data/price-change-journal.csv` в workspace категорийного менеджера.

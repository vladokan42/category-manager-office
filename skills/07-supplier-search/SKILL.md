# Поиск и оценка новых поставщиков

Поиск потенциальных поставщиков по категории/региону/условиям и оценка их привлекательности.

## Сценарии использования

- «Найди поставщиков труб ПНД в ЦФО»
- «Оцени Поставщика Х — надёжный?»
- «Подбери 3-5 поставщиков для категории "запорная арматура"»

## Входные данные

- Категория товара или список позиций
- Критерии: регион, минимальная партия, отсрочка
- Известные конкуренты/ориентиры

## Действия

1. Поиск в интернете (web_search) контактов поставщиков в заданной категории
2. Для каждого кандидата собрать:
   - Наименование, сайт, контакты
   - Ассортимент (основные группы)
   - Регион, условия (если указаны на сайте)
3. Оценить по шкале:
   - Полнота ассортимента
   - Ценовой сегмент (если есть данные)
   - Регион (ближе = лучше)
   - Срок работы на рынке
4. Ранжировать кандидатов, выдать топ для первого контакта
5. При необходимости — сформировать шаблон запроса КП

## Ограничения

- Анализ по открытым источникам, без коммерческих баз
- Финальное решение — за пользователем

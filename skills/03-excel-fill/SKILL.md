# Заполнение Excel

По таблице с артикулами найди последнюю закупочную цену из базы и подставь в нужную колонку.

## Сценарии использования

- «Вот файл с артикулами, заполни колонку D последними закупочными ценами»
- «Добавь к таблице колонки: поставщик, дата последней закупки, цена»

## Входные данные

- Excel-файл с артикулами (минимум одна колонка с артикулами)
- База закупочных цен (история из 1С, CSV, другой Excel)

## Действия

1. Загрузить файл пользователя
2. Определить колонку с артикулами (спросить, если неочевидно)
3. Определить целевую колонку для заполнения
4. Для каждого артикула:
   - Найти последнюю закупочную цену (по max дате)
   - Подставить в указанную колонку
   - Если цены нет — оставить пустой или «нет данных»
5. Вернуть заполненный файл

## Источники цен

- История закупок из 1С (выгрузки на почту, навык `1c-integration`)
- Вручную загруженные CSV/Excel
- Файлы в workspace

# Поиск FMCG-дистрибьюторов и товаров (РФ)

Поиск поставщиков и конкретных товаров (например, Dove, Nivea, Coca-Cola и т.д.) в Москве, Санкт-Петербурге и других регионах РФ.

## Сценарии

- «Найди гель для душа Dove в Москве — кто возит?»
- «Кто дистрибьютор [бренд] в [регион]?»
- «Найди [товар] у поставщиков — цена, контакты»

## Этапность

### 1. Определить производителя / бренд-холдинг

До поиска дистрибьюторов — понять, кто производит бренд:

| Бренд | Холдинг |
|---|---|
| Dove, Rexona, Axe, Lipton, Knorr, Hellmann's, Domestos, Cif | Unilever |
| Nivea, Eucerin, Hansaplast, Labello, Florena | Beiersdorf |
| Pampers, Ariel, Tide, Gillette, Head&Shoulders, Pantene | Procter & Gamble |
| Coca-Cola, Sprite, Fanta, BonAqua, Powerade | Coca-Cola HBC / Coca-Cola Europacific Partners |
| Pepsi, Lay's, 7Up, Mirinda, Aqua Minerale | PepsiCo |
| Mars, Snickers, Milky Way, Pedigree, Whiskas, Royal Canin | Mars Inc |
| Nestlé, Nescafé, Nuts, KitKat, Maggi, Purina, Gerber | Nestlé |

Если бренд не угадывается — ищем через web_search: `[бренд] производитель холдинг`.

### 2. Поиск дистрибьюторов (официальных / неофициальных)

#### А. Официальные дистрибьюторы
Запросы (использовать `tavily_search` с `search_depth=advanced`):

```
дистрибьютор [холдинг/бренд] Москва
официальный дистрибьютор [бренд] Санкт-Петербург
[бренд] дистрибьюторская сеть Россия
дилер [бренд] опт
```

Проверить сайт производителя — обычно есть раздел «Где купить» / «Дистрибьюторы».

#### Б. B2B-площадки
Запросы:

```
site:pulscen.ru [бренд] [товар]
site:tiu.ru [бренд] [товар]
site:optom-plus.ru [бренд] [товар]
site:satu.kz [бренд] [товар] (Казахстан — если товар идёт через РК)
[бренд] [товар] оптом Москва
```

Для каждой находки — извлечь:
- Название компании
- Контакты (телефон, email, сайт)
- Цену (если указана)
- Регион

#### В. Параллельный импорт / серый рынок
Если официальных дистрибьюторов нет (бренд ушёл из РФ) — запросы:

```
[бренд] параллельный импорт Москва
[бренд] [товар] опт неоригинал
```

Уточнить у пользователя, интересует ли его только официальный канал.

### 3. Сбор контактов

Для каждого кандидата собрать в таблицу:

| Компания | Контакты | Регион | Ассортимент | Цены (если есть) | Источник |
|---|---|---|---|---|---|

Использовать `tavily_extract` для извлечения контактов со страниц-кандидатов.

### 4. Оценка кандидата

| Критерий | Вес | Как оценить |
|---|---|---|
| Регион | 30% | Москва/СПб → 10, ЦФО → 7, остальное → 5 |
| Ассортимент | 25% | Совпадает с запросом → 10, частично → 5 |
| Срок работы | 20% | 5+ лет → 10, 2-5 → 7, <2 → 4 (искать через историю домена или ИНН) |
| Официальность | 25% | Прямой дистрибьютор → 10, дилер → 7, перекуп → 3 |

Если есть ИНН — можно проверить через СБИС/Контур.Фокус строкой `site:sbis.ru [ИНН]` (откроется выписка).

### 5. Формирование результата

Выдать пользователю:
1. **Таблица кандидатов** с оценкой
2. **Контактные данные** для первого звонка/письма
3. **Холдинг-производитель** (если определён)
4. При необходимости — **шаблон запроса КП**

## Шаблон запроса КП для первого контакта

```
Тема: Запрос коммерческого предложения — [товар/категория]

Здравствуйте!

Меня зовут [имя], я категорийный менеджер ООО ТД «ЛФБ» (Москва).

Прошу направить коммерческое предложение на поставку [товар/категория]:
- Артикулы (если есть)
- Закупочные цены (с НДС и без)
- Минимальная партия
- Условия поставки (самовывоз / доставка до Москвы)
- Сроки отгрузки

Буду признателен за оперативный ответ.

С уважением,
[имя]
[контакты]
```

## Примечания

- Неофициальные поставщики могут быть дешевле, но нет гарантии качества
- Для точного поиска конкретного артикула — использовать также `article-search` skill (если доступен)
- Если нужна замена товара (срыв поставки) — использовать `supplier-substitution-search` skill

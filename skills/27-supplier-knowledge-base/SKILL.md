# База знаний поставщиков

Структурированное хранилище информации о каждом поставщике: контакты, менеджеры, история изменений, условия работы, заметки.

## Сценарии использования

- «Покажи карточку Поставщика Х»
- «У Поставщика Y сменился менеджер, обнови»
- «Какие поставщики работают с отсрочкой 30+ дней?»
- «Сводка по всем активным поставщикам»

## Структура карточки

```
Поставщик:
  Наименование (полное/краткое)
  Контакты: телефон, email, сайт
  Менеджеры (текущий + история с датами):
    - ФИО, должность, телефон, email, дата начала работы
  Реквизиты (текущие + история изменений):
    - ИНН, КПП, банк, расчётный счёт
  Условия работы (с датой актуальности):
    - Отсрочка, мин. партия, доставка, возврат брака
  Рейтинг пользователя: 1-5
  Заметки: свободный текст
```

## Действия

1. Поиск поставщика по названию/ИНН/контактам
2. Создание/редактирование карточки
3. Автоматическое обновление из писем (навык `email-data-extraction`)
4. Вывод сводки по всем поставщикам
5. Экспорт/импорт

## Хранение

Файл `data/suppliers.json` или SQLite в workspace категорийного менеджера.

## 🔴 Критические правила (коротко)

**1. Не отмечать письма прочитанными** — только `BODY.PEEK[]`, `BODY[]` запрещён.
**2. Не разглашать цены** — только доверенным внутренним адресатам.

Подробно → [[memory/_Критические_Правила]]

## 🏆 Главные принципы

### Процесс работы (утв. 26.06.2026)
1. Сначала вопросы, потом автономная работа до 100%
2. После старта — никаких промежуточных возвратов
3. Возврат только при готовом результате или объективном блокере

### Не плодить скрипты
Прямые действия лучше многофайловых решений.

## 📝 Стиль переписки категорийного менеджера

Полный портрет → `reference/vlad_writing_style.md`
Ключевые правила для писем:
- Приветствие по адресату (близкие → «[Имя], привет», коллеги → «Коллеги, добрый день»)
- Дата в теме: YYYY/MM/DD
- Минимум воды, сразу суть
- Дефисные списки при перечислении
- Таблица цен: артикул, наименование, НДС, без НДС, с НДС
- Знак рубля — ₽
- Вывод/вердикт обязателен
- Подпись: полная (для внешних), короткая (для коллег)

## 📊 Формат выдачи при поиске товара

При выдаче предложений по товарам из реестра:

```
Поставщик                    | Артикул       | Описание                     | Реестр,₽ | Разн,₽ | Дата
```

- Столбцы: Поставщик | Артикул | Описание (коротко, влезает в строку) | Реестр,₽ | Разн,₽ | Дата
- В заголовке указывать «наша ХХ,₽»
- Сортировка по Разн,₽ (отрицательные → положительные)
- Все цифры в одну строку без переносов
- Моноширный блок (код-блок ```)

## 📋 Практика ответов на письма (сессия 23.06.2026)

### Процесс
1. Показать в чате: письмо (№, от кого, тема) + короткий контекст + вариант ответа
2. Утверждение категорийным менеджером
3. После одобрения — черновик в IMAP (папка «Черновики», флаг \Draft, категория [AI])

### Формат ответа в чате
```
📧 Письмо №N — [От кого] [время]
*Тема: [тема]*

[Контекст — о чём письмо]

[Адресация]. [Суть — 1-2 предложения].
```

### Ключевые правила (из практики)
- **Не упоминать других поставщиков** в ответе про одного поставщика
- **Не использовать длинные дефисы/тире** — только короткий дефис (-)
- **Письма, на которые уже дан ответ, пропускать**
- **Письма по поставщикам не из утверждённого списка пропускать**
- **Вопросы по срокам годности/ОСГ адресовать в отдел закупок**
- **Если письмо информационное/благодарность — пропускать**

### Черновик (IMAP)
- From НЕ ставить — Outlook подставит
- Keywords: [AI] обязательно
- In-Reply-To + References с Message-ID исходного письма
- Подпись корпоративная (для коллег — сокращённая)
- Папка черновиков: `&BCcENQRABD0EPgQyBDgEOgQ4-`
